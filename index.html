Custom CSS3 grid designed as substitute to utilising grids from other frameworks (Bootstrap,Foundation,etc...). This grid system is responsive and easily customizable for my project needs.

Instructions:

1. Download grid.css from git repository


2. Add following markup to your html:


<div class="row">
    <div class="col one"> content goes here </div>
    <div class="col twelve"> content goes here </div>
</div>
